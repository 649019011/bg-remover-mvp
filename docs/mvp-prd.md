## 1. 产品概述

### 1.1 产品定位
一款基于 AI 的在线背景去除工具，帮助用户快速将图片背景透明化，专注于海外市场。

### 1.2 目标市场
- **主要市场：** 美国、英国（英语市场，验证期）
- **次要市场：** 加拿大、澳大利亚、新西兰
- **扩展市场：** 东南亚（印尼、泰国、越南）、拉美（巴西、墨西哥）

### 1.3 目标用户
| 用户类型 | 需求场景 | 使用频率 | 付费意愿 |
|---------|---------|---------|---------|
| 电商卖家 | 产品图去背景（Amazon、eBay、Shopify） | 高 | 高 |
| 社交媒体创作者 | Instagram、TikTok 内容制作 | 中 | 中 |
| 设计师 | 快速抠图工作流 | 高 | 高 |
| 普通用户 | 证件照、头像处理 | 低 | 低 |

## 2. MVP 核心功能

### 2.1 必须功能 (P0)

#### 2.1.1 图片上传
- 支持拖拽上传
- 支持点击选择文件
- 支持的格式：JPG、PNG、WebP
- 最大文件大小：10MB
- 上传预览

#### 2.1.2 背景去除
- 调用 remove.bg API 处理
- 显示处理进度（Loading 状态）
- 错误提示（网络错误、文件格式错误、API 限额）

#### 2.1.3 结果展示
- 原图 vs 处理后图片对比
- 支持放大/缩小预览
- 透明背景显示（棋盘格背景）

#### 2.1.4 下载
- 下载为 PNG 格式（透明背景）
- 文件名：`removed-background-[timestamp].png`

### 2.2 重要功能 (P1)
- 多语言支持（英语、中文）
- 响应式设计（移动端适配）
- 使用次数限制提示（免费 5 张/月）
- API 状态显示（remove.bg 额度）

### 2.3 未来功能 (P2)
- 批量处理
- 背景替换（纯色、渐变、自定义图片）
- 不同格式输出（JPG、WebP）
- 边缘调整（羽化、透明度）
- 历史记录

## 3. 技术架构

### 3.1 技术栈

**前端：**
- 框架：React + Vite
- UI：Tailwind CSS
- 部署：Cloudflare Pages

**后端：**
- Cloudflare Workers（Serverless）
- API 集成：remove.bg API

**域名与 CDN：**
- Cloudflare DNS
- 免费 SSL 证书

### 3.2 系统架构图

```
用户浏览器
    ↓
Cloudflare Pages (前端)
    ↓
Cloudflare Worker (后端)
    ↓
remove.bg API
```

### 3.3 数据流

1. 用户上传图片到 Cloudflare Worker
2. Worker 验证文件格式和大小
3. Worker 转发请求到 remove.bg API
4. remove.bg 返回处理后的图片（PNG）
5. Worker 返回图片给前端
6. 前端展示并允许下载

**特点：**
- 无需存储图片（纯内存处理）
- 无用户数据库（MVP 阶段）
- 无状态设计（可扩展）

## 4. 非功能需求

### 4.1 性能要求
- 图片处理时间：< 5 秒（90% 情况下）
- 页面加载时间：< 2 秒
- 并发支持：至少 10 个同时处理请求

### 4.2 可用性
- 系统可用性：99.5%
- API 容错：remove.bg 服务异常时友好提示
- 错误处理：所有操作都有明确的错误反馈

### 4.3 安全性
- CORS 配置（仅允许本域名调用）
- API Key 加密存储（Cloudflare 环境变量）
- 文件类型验证（防止恶意文件上传）
- 文件大小限制（10MB）

### 4.4 浏览器兼容性
- Chrome 90+
- Safari 14+
- Firefox 88+
- Edge 90+
